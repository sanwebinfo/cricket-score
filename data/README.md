## Built Using

- PHP
- Silm Framework
- symfony DOM crawler and CSS selector
- Guzzle - PHP HTTP client
- Composer for install and Manage PHP packages  

## Setup

- Download this repo or clone
- Goto folder `data` and install required packages

```sh
cd data
composer install
```

- Testing

```sh
composer start
```
